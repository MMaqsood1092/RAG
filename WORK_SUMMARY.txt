QUERY FLOW ARCHITECTURE

User Question
    ↓
[HuggingFace] Query Expansion (2-3s)
    ↓
Route Decision:
├─ SQL Path → [HuggingFace] SQL Gen (3-4s) → Execute
├─ Attachment Path → [HuggingFace] QA (5-8s)
├─ Events Path → Direct DB query
└─ RAG Path → Vector Search (1-2s) → [HF] Response (5-8s)
    ↓
Response to User (10-15s total)

KEY CHANGES

OLD (OpenAI)
├─ import OpenAI from "openai"
├─ const client = new OpenAI(...)
├─ await client.chat.completions.create()
└─ Cost: $0.10/query

NEW (HuggingFace + Voyage)
├─ import { generateChatResponseHF } from services
├─ await generateChatResponseHF(prompt, question, context)
├─ Voyage for embeddings (kept - best quality)
└─ Cost: $0.00005/query (99.95% savings)
